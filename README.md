# Biblioteca System

Proyecto final — sistema simple de gestión de biblioteca implementado con Python y Flask.

Estructura principal:

```
biblioteca-system/
├── app/
├── tests/
├── config.py
├── requirements.txt
└── database.sql
```

Instrucciones rápidas:

1. Crear y activar un entorno virtual (Mac zsh):

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. Ejecutar la app:

```bash
export FLASK_APP=app.main
flask run
```

3. Acceder en http://127.0.0.1:5000
